getRelativeDate(year, month, date, dateObj) → {Object}
utils.js, line 129

Get related date hash with parameters that include date information.

Name Type Description
year number

A related value for year(you can use +/-)
month number

A related value for month (you can use +/-)
date number

A related value for day (you can use +/-)
dateObj Object

Returns:

dateObj

Example

 utils.getRelativeDate(0, 0, -1, {year:2010, month:1, date:1}); // {year:2009, month:12, date:31}
